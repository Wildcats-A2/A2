Cloud, Services & Servers

What does it do?

Clouds are servers that provide different services such as the running of software and applications, accessing and storing data virtually via the internet. This means that we do not have to rely on the physical hard disk to do these jobs. Many tasks could not be accomplished without the existence of a cloud server. Clouds also allow many end users to access the same file or application on different devices at different locations and time as long as they are connected to the internet. In addition, cloud computing enables scaling up and down to be changed easily with much lesser cost according to the user's needs and requirements. Therefore, it is normally fully utilised by the end users.

There are 3 types of cloud computing which are public, private and hybrid:

•Public - is when the server is owned by the cloud service provider. All hardware and software are fully provided by the “owner” of the server which explains why the public cloud computing costs the lowest as the purchase of infrastructure and software is not required by the user. Some public clouds are even provided for free.Organisation or individuals will be sharing the same hardware, storage and network devices.( Azure n.d.)Examples of public clouds are Gmail, Amazon Web Services (AWS) and Google Drive.

•Private - can only be used by only 1 organisation and it cannot be shared with others. The organisation will need to have their own hardware and network and hence it incurs high operation cost. In contrast, a private cloud provides a higher level of security as compared to a public cloud. Examples of private clouds are Oracle, Microsoft and HP Data Centers.

•Hybrid - the combination of both private and public clouds which depends on the customer’s current requirements and needs. Examples of hybrid clouds are Amazon Web Services (AWS), Microsoft Azure and Google Cloud Platform(GCP).

What can be done now?

3 Types of cloud services:

IaaS [ Infrastructure as a Service] - Cloud provider is in charge of building the infrastructure while the user manages both the software and platform.Eg. DigitalOcean, Linode, Rackspace, Amazon Web Services (AWS), Cisco Metapod, Microsoft Azure, Google Compute Engine (GCE)

PaaS [ Platform as a Service] - Cloud provider manages the infrastructure and the platform while the user manages only the software. It is a platform for developers who want to build their own applications without having to worry about the infrastructure. Eg. AWS Elastic Beanstalk, Windows Azure, Heroku, Force.com, Google App Engine, Apache Stratos

SaaS [ Software as a Service] - Cloud provider manages the infrastructure, platform and software. In SaaS, a user is not required to download or install the software on the personal computer as it could also be accessible via web browser. Eg.Google Docs, Outlook, Gmail

What is likely to be able to do be done in the future?

In the near future, it is very likely that cloud technologies will be applied to most home appliances namely washing machines, microwave ovens, kettles and household garbage bins.

For instance, when the household garbage bin sensor detects the rubbish is already full, it will send notification to the waste department to collect the garbage bin. This could avoid sending out a garbage truck on a regular basis when it is not required to save cost and time.

Another example would be the “smart” microwave oven that requires cloud technologies to ensure that the microwave oven could make accurate predictions of the cooked food when it is ready to be served. The microwave oven will then automatically turn off the heat.

From a health perspective, the invention of smart watches can also be an example. The future smart watches should be able to link an individual's health conditions with the hospital which allows the hospital to keep track of the individual’s well-being. They could also be reminded to go for a regular health check-up. For instance, let’s say there’s a sudden heart attack happens to any individual, the smart watch would be able to send out alert messages or signals to the emergency department for immediate action.

What technological or other developments make this possible?

The main technology used in the cloud is virtualisation. Virtualisation is a process of creating software and virtual based of storage, compute, networking, servers or applications. Virtualisation permits the increase in scalability, efficiency and cost effectiveness in building a cloud technology.

To set up a cloud infrastructure, we also need both hardware and software which work hand in hand. The hardware includes networking equipment, such as switches, routers, firewalls, and load balancers, storage arrays, backup devices, and servers. (RedHat 2019)

Software like hypervisor is run above the physical server. It functions as a “transport” that abstract resources from the physical server and allocate them to the virtual environment. (IBM 2019)

Building a cloud requires a significant number of physical computers that will host several virtual systems. Cool equipment is also installed to avoid overheating.

What is the potential impact of this development?

The first implication of clouds is that they enable everyone to work remotely as long as the users are connected to the internet. It increases the efficiency of collaborations as collaborations of projects are made possible with people of different nations.

In addition, employees will be able to work in a flexible environment. For some jobs, employees are not required to be physically present at the workplace. They will be able to complete the tasks assigned from home.

What is likely to change?

As cloud technology is improving drastically, it has changed and eased our lives in various aspects. Security and privacy of the cloud is always the major concern. Top priority has to be considered for the improvement of privacy and security to ensure that data of the end users are 100% well protected.

Apart from security and privacy, there might be a possibility that some hardware components will be replaced by cloud. In the future, we might be using low-power processors to maximise the battery lifespan corresponding with a powerful cloud that costs way cheaper than investing on a better processor.

Which people will be most affected and how?

Businessmen because they no longer require to purchase expensive systems and hardware in order to run their businesses. They can just simply make use of resources provided by the cloud computing service provider. This will help to reduce most of the operating cost. Instead, they can use the money to build their revenue.

Furthermore, with the support of cloud computing, it allows businesses to scale up or down depending on the current needs and requirements. They can make their purchases on cloud storage for an ongoing project which requires a big volume of storage to store files and data. Once the project is done and such a large amount of storage is no longer required, they can cease the purchase of cloud storage.

Will this create, replace or make redundant any current jobs or technologies?

Yes, clouds will eventually create and replace jobs to a certain extent. IT jobs that cloud computing will create are applications architectures, software engineers and developers, financial analysts, etc.

On the contrary, since cloud technology remotely centralises networks, positions that are related to system admission or customer services might be entirely replaced by the cloud.

When people start to depend solely on the cloud, they will not have to build their own infrastructure. This would mean that the demand for hardware will decrease since there will be a shift from hardware to services. Manufacturers that manufacture hardware will be threatened by the cloud as they will be making less profit than before.

In your daily life, how will this affect you?

Cloud servers have become indispensable in everyone’s life which most people are still unaware of. Majority of the activities that are carried out via our gadgets require cloud computing technologies.

I often use social media platforms such as Facebook, Twitter and Tiktok to keep in touch with friends and families. Youtube and Spotify to stream videos and listen to music. They have unintentionally become one of my favourite pastimes.

Besides that, being a person who has the tendency to misplace things especially when they are small and easily neglected, cloud storage is one of the best solutions as it could replace pen drives or hard disks. I can simply store data or files on google drive which allows me to access it anywhere as I wish and without having to worry about misplacing my pen drives / hard disk again.
